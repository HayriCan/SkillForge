# Views

Skill Forge organizes your Claude Code configuration into 12 distinct views, each focused on a specific resource type.

## Resource Views

| View | Description |
|------|-------------|
| **Agents** | Create and edit agent markdown files (`~/.claude/agents/`). |
| **Skills** | Manage skill definitions that extend Claude Code's capabilities. |
| **Commands** | Define custom slash commands for your workflows. |
| **Hooks** | Configure event-driven hooks that trigger on specific actions. |
| **Plans** | Browse and manage plan files created during Claude Code sessions. |
| **Plugins** | Enable, disable, and configure plugins. |
| **Sessions** | View session history and related task files. |
| **Tasks** | Browse task session files in JSON and Markdown formats. |
| **Teams** | Manage team configuration files. |
| **Todos** | View and manage todo items from Claude Code sessions. |
| **Config** | Tree view of all `CLAUDE.md` and `MEMORY.md` files across project subdirectories. |
| **Settings** | Structured editor for `settings.json` — environment variables, permissions, MCP servers, and preferences. |

## Common Features

Every view shares a consistent set of features:

- **File list** with search and sort
- **Syntax-highlighted editor** for Markdown and JSON files
- **Inline validation** that catches errors before you save
- **Diff preview** showing exactly what changed
- **Version history** to restore previous file states
- **Empty state** with guidance when no files exist yet
