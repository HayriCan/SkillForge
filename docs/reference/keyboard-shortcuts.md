# Keyboard Shortcuts

Skill Forge supports a full set of keyboard shortcuts for fast navigation and editing.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd K` | Open global search / command palette |
| `Ctrl/Cmd S` | Save current file |
| `Ctrl/Cmd N` | New file in active view |
| `Ctrl/Cmd Z` | Undo last edit |
| `Ctrl/Cmd Shift Z` | Redo |
| `Escape` | Close modal / palette |

## Navigation

| Shortcut | Action |
|----------|--------|
| `1` | Switch to Agents view |
| `2` | Switch to Skills view |
| `3` | Switch to Commands view |
| `4` | Switch to Hooks view |
| `5` | Switch to Plans view |
| `6` | Switch to Plugins view |
| `7` | Switch to Tasks view |
| `8` | Switch to Config view |
| `9` | Switch to Settings view |

## Accessibility

| Shortcut | Action |
|----------|--------|
| `Ctrl =` | Increase font size |
| `Ctrl -` | Decrease font size |
| `Tab` | Move focus to next element |
| `Shift Tab` | Move focus to previous element |
